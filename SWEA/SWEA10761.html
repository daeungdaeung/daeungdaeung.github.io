이 문제는 엄청 어려운 것은 아니지만, 나처럼 코딩을 제대로 배운적이 없고 경험이 적은 사람에게는 꽤나 까다로운 문제였다. (물론 저같은 상황에서도 배움이 빠르거나 센스있는 사람이라면, 무난하게 풀었을것 같아요 ㅎ ㅎ)

내 생각은 이쯤에서 그만하고, 문제 풀이를 확인해보자.

아이디어

첫번째 테스트 케이스로 설명을 하고자 한다.

아래 그림은 데이터를 저장한 모습을 보여준다. 대략 설명을 하자면, Blue가 눌러야하는 버튼과 Orange가 눌러야하는 버튼을 개별 리스트에 저장한다.

아래 그림들은 내가 작성한 코드가 어떻게 동작하는지 그 흐름을 설명해준다.

처음에는 모든 인덱스가 0으로 초기화되었다. A_idx가 아직 첫번째를 가리키고 있으므로 Orange는 눌러야할 버튼(1번 버튼) 앞에 서 있지만, 아무 동작도 하지 않는다. Blue는 계속해서 버튼(2번 버튼) 방향으로 움직이고 결국 해당 버튼에 도착하면 버튼을 누른다.

위에서 Blue가 버튼을 눌렀으므로 B_idx와 A_idx는 오른쪽으로 한칸씩 움직인다.

이어서 Orange도 버튼(1번 버튼)을 누르기 때문에 A_idx와 O_idx도 오른쪽으로 한칸씩 움직인다.

바로 위 그림에서, A_idx가 (O, 2)를 가리키고 있으므로 Orange가 먼저 작동한다. Orange가 2번 버튼을 누르면 Orange는 더이상 누를 버튼이 없으므로 동작이 종료된다.

위에서와 동일하게 A_idx와 O_idx는 한칸씩 움직이고 아래 그림과 같이 표현될 수 있다.

지금까지 인덱스를 활용하여, 주어진 버튼 순서대로 어떻게 구현하면 되는지 그 흐름을 간단하게 설명했다. 하지만 코드 작성은 더 세밀한 부분들이 존재하고 그 부분들도 다루어 주어야한다. (Blue와 Orange의 position을 계속 옮기는 것 등)

큰 흐름만 이해하고 자세한 부분들은 코드를 뜯어보면서 직접이해하면 좋을 것 같다. 주석을 최대한 작성해보았으니 주석 참고 바란다.

Pycharm의 디버그를 활용하면 변수 값들이 어떻게 변하는지 볼 수 있다. 변하는 값들을 추적하면서 코드를 보면 더 잘 이해가 될 것이라 생각한다.

import collections

T = int(input())
for tc in range(1, T+1):
tmp = list(input().split())
N = int(tmp[0])
data = tmp[1:]

A_data = collections.deque([])  # 모든 데이터를 저장 -> Blue 또는 Orange가 버튼을 누르는 순서
B_data = collections.deque([])  # Blue가 눌러야하는 버튼 저장
O_data = collections.deque([])  # Orange가 눌러야하는 버튼 저장

for i in range(0, N*2, 2):
    if data[i] == 'B':
        B_data.append(int(data[i+1]))
    else:
        O_data.append(int(data[i+1]))
    A_data.append((data[i], data[i+1]))

B_position = 1
O_position = 1
B_idx = 0
O_idx = 0
A_idx = 0
len_B_data = len(B_data)
len_O_data = len(O_data)
len_A_data = len(A_data)
time = 0
# 전체 데이터를 순회하였으면, while문 종료
while A_idx < len_A_data:
    # while문이 한번 도는 것은 단위시간이 지남을 의미
    # 버튼을 누르면 bt가 True
    bt = False
    # B_idx가 B_data 크기를 벗어나면 아무것도 실행하지 않음
    if B_idx < len_B_data:
        # B의 위치(B_position)가 눌러야하는 버튼 위치보다 작은 경우 -> +1
        if B_data[B_idx] - B_position > 0:
            B_position += 1
        elif B_data[B_idx] - B_position < 0:
            B_position -= 1
        else:  # B의 위치가 눌러야하는 버튼 위치일 경우
            # 현재 Blue가 버튼을 눌러야하는 차례라면
            if A_data[A_idx][0] == 'B':
                # 버튼을 누르면 다음에 어떤 버튼을 눌러야하는지 알아야한다.
                # 버튼을 눌렀음을 알려주는 bt
                bt = True
                # 블루 차례이므로 Blue가 버튼을 누르고 다음에 눌러야 하는 버튼 idx로 이동
                B_idx += 1
            # Blue가 버튼을 누르는 차례가 아니라면 아무것도 수행 X
    if O_idx < len_O_data:
        if O_data[O_idx] > O_position:
            O_position += 1
        elif O_data[O_idx] < O_position:
            O_position -= 1
        else:
            if A_data[A_idx][0] == 'O':
                bt = True
                O_idx += 1
    if bt:
        A_idx += 1
    time += 1

print('#{} {}'.format(tc, time))
